# TOUCHBeauty Colombia — Assets Repository

Assets de marca, fotos de producto, certificaciones y briefs de diseño para **TOUCHBeauty Colombia** (`@touchbeautycol`).

> **Uso con Claude Design:** copia la URL raw de cualquier imagen y pégala en el prompt — Claude Design la descarga directamente sin necesidad de subir archivos.

---

## Estructura

```
touchbeauty-assets/
├── brand/          → Logo y assets de identidad visual
├── productos/      → Fotos principales por producto (fondo blanco)
├── briefs/         → Prompts y briefs para Claude Design
├── manuales/       → Manuales PDF de productos
└── MATERIAL/       → Material completo por modelo (fotos, certificaciones)
    ├── 1760/       → Cepillo Facial AQUA (TB-1760)
    ├── 1767/       → TRIFECTA (TB-1767)
    ├── 1769/       → Espátula Ultrasónica (TB-1769)
    ├── 1859/       → Rizador de Pestañas de Silicona (TB-1859)
    ├── 2155/       → IPL (TB-2155)
    ├── 2187/       → CANC — Dispositivo Facial 6 en 1 (TB-2187)
    ├── 2343/       → VITA Peine Multifuncional (TB-2343)
    ├── 2386/       → GLAM LED Facial Mask (TB-2386)
    ├── 2443/       → Cepillo Capilar (TB-2443)
    └── 2546/       → VITA Graphene Casco (TB-2546)
```
