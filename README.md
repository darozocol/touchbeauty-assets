# TOUCHBeauty Colombia — Assets Repository

Assets de marca, fotos de producto y briefs de diseño para **TOUCHBeauty Colombia** (`@touchbeautycol`).

> **Uso con Claude Design:** copia la URL raw de cualquier imagen y pégala en el prompt — Claude Design la descarga directamente sin necesidad de subir archivos.

---

## Estructura

```
touchbeauty-assets/
├── brand/       → Logo y assets de identidad visual
├── productos/   → Fotos oficiales fondo blanco de cada dispositivo
└── briefs/      → Briefs de diseño para tarjetas de regalo y materiales
```
